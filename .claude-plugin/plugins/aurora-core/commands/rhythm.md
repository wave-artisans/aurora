# Rhythm Command

Analyze your color patterns over time.

## Usage

```
/rhythm [daily|weekly|monthly]
```

## Description

The rhythm command visualizes your Aurora mode patterns over time, helping you see your energy distribution as a waveform.

## Monthly Rhythm

```
/rhythm monthly
```

Shows:
- Week-over-week color trends
- Mode that's growing/shrinking
- Burnout indicators
- Recommendations

## Daily Rhythm (Today)

```
/rhythm daily
```

Shows:
- Hour-by-hour activity
- Current mode
- Suggested transitions

## Philosophy

Aurora is about rhythm, not rigid schedules. The rhythm command helps you:
- See your natural patterns
- Identify imbalances before burnout
- Celebrate good balance

## Related Commands

- `/surf-report` - Today's energy check
- `/aurora-status` - Current balance
- `/rebalance` - Specific recommendations
